My Coffee House

Front-end web application that includes search, shopping cart and checkout functionalities.

Project Overview

My Coffee House is a fictional coffee shop website that started as a mid term project during my first semester in the Interactive Media Design program at Algonquin College. The goal was to create a modern, accessible, and visually engaging website using just HTML and CSS. Over time, I enhanced the project by integrating Tailwind CSS, adding interactivity with JavaScript, and improving the overall user experience.

Goal

The main goal of this project was to create a user friendly and visually appealing coffee shop website that allows users to browse the menu, add items to their cart, and proceed to checkout. Additionally, I aimed to enhance my skills in HTML, CSS, JavaScript, and Tailwind CSS while ensuring the site is accessible and responsive across different devices.

Process

Throughout the development process, I encountered a few challenges:

  1. Challenge: Cart data resets when the page loads.
    Solution: I used localStorage to save and retrieve cart data. This ensures that a user's selections persist even if they refresh the page or navigate away and come back.
  2. Challenge: Page not being loaded because of missing elements.
    Solution: I used an if() statement to check if the required elements exist on the page then execute the logics.
  3. Challenge: Implementing a search feature to filter menu items dynamically.
    Solution: I used addEventListener() to add an event then use the JavaScript array method forEach() to loop through each menu item while includes()check if the search value is found in the menu items container.
  4. Challenge: Dynamically updating the cart total and item count as items were added or removed.
    Solution: I created functions to calculate the total price and item count from the data stored in localStorage. These functions are called every time the cart is modified, and they update the corresponding elements in the DOM to provide immediate feedback to the user.
